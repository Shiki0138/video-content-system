VideoAI Studio セットアップガイド

1
🔄 リポジトリをクローン
GitHubからVideoAI Studioのソースコードをダウンロードします
git clone https://github.com/Shiki0138/video-content-system.git cd video-content-system
2
🐍 Python環境の確認
Python 3.8以上がインストールされていることを確認します
python3 --version # もしPython3がインストールされていない場合（macOS）: brew install python3
3
📦 自動セットアップ実行
必要なライブラリとWhisperモデルを自動インストールします
python3 setup.py
⚠️ 初回実行時はWhisperモデルのダウンロードで数分かかる場合があります
4
⚙️ 設定ファイルの確認
config.yamlファイルで各種設定を調整できます（オプション）
nano config.yaml # または code config.yaml
5
🚀 WebUIを起動
ブラウザベースのUIを起動してVideoAI Studioを使用開始
python3 web_app.py
🌐 http://localhost:8003 で起動

  • 📝 BlogOptimizer AIによる高品質ブログ生成
